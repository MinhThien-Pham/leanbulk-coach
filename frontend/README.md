# LeanBulk Coach Frontend

This is the React + Vite frontend application for the LeanBulk Coach demo.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Configuration

By default, the application connects to the backend at `http://localhost:8000`. You can configure this by setting the `VITE_API_BASE_URL` environment variable.
